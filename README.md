# 早期打卡 hitcard
间基于Vue.js、Vue Router、Egg.js、MongoDB等开发了此 H5游戏。用户每天支付1元参与打卡，第二天早上五点到八点打卡随机瓜分奖池里的现金。既可以锻炼早起，又可以抢到红包 

## 目录结构

```
  . hitcard
  ├── app                       # 前端
  │   ├── src                   # 源文件
  │   │   ├── assest            # 图像资源
  │   │   ├── components        # 组件
  │   │   └── router            # 路由
  │   ├── static                # 静态资源目录
  │   ├── config                # 配置文件
  │   └── build                 # 打包文件
  └── server                    # 后端
      ├── src                   # 源文件
      │   ├── controller        # 控制器
      │   ├── model             # 数据库 Schema
      │   ├── public            # 静态资源目录
      │   └── schedule          # 定时器代码
      ├── config                # 配置文件
      └── test                  # 测试

```
